# Loyalty Cards Wallet

A simple Android application to store and manage loyalty/mech cards (Tesco Clubcard, etc.) with persistent storage using Room database.

## Features

- Add loyalty cards with name, card number, and optional notes
- Edit existing cards
- Delete cards
- All cards are persisted in a local Room database
- Material Design 3 UI
- Clean architecture with ViewModel and Repository pattern

## Project Structure

This is a complete Android Studio project. Simply open it in Android Studio and run.

## How to Use

1. Open Android Studio
2. Select "Open" and navigate to this project directory
3. Wait for Gradle to sync
4. Run on an emulator or physical device

## Package Name

`com.jv.loyaltycardswallet`

## Dependencies

- AndroidX Core KTX
- AndroidX AppCompat
- Material Design Components
- AndroidX ConstraintLayout
- AndroidX RecyclerView
- AndroidX Lifecycle (ViewModel, LiveData)
- AndroidX Room (for database)
